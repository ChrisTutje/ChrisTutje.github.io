Functionality

Return of the Fallen is a small turn-based RPG with an overworld, battlesystem, and original soundtrack. It was inspired by classic RPG's like Final Fantasy.

Dependencies

Return of the Fallen was built in the Unity game engine, which is based in C#. GitHub was used as version control. The sound effects and instruments were created from scratch using Vital, and the music was arranged using FL Studio as a DAW.

Group

Return of the Fallen was created by team Pixel Plunderers over the course of 6 weeks for the first TechWise cohort II group project. It consisted of Eric Holman as the lead UI designer and battle animator, Amber Miller as the cutscene designer, Jaya Sicard as the overworld designer, and Adam Ahmetasevic as character movement.

My Role

For this project I served as the project leader. I had the initial idea of creating an RPG, and so I was the defacto creative designer. I designed the back-end logic for the battle-system, helped design some of the menu UI, and I composed an original soundtrack creating custom sounds using a digital synthesizer.
